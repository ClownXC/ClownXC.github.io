    kafka manager工具安装

    为了简化开发者和服务工程师维护 Kafka 集群的工作，yahoo 构建了一个叫做 Kafka 管理器的基于 Web 工具，Kafka Manager。这个管理工具可以很容易地发现分布在集群中的哪些 topic 分布不均匀，或者是分区在整个集群分布不均匀的的情况。支持管理多个集群、选择副本、副本重新分配以及创建 Topic。同时，这个管理工具也是一个非常好的可以快速浏览 Kafka 集群的工具

    1.kafka-manager介绍

    为了简化开发者和服务工程师维护 Kafka 集群的工作，yahoo 构建了一个叫做 Kafka 管理器的基于 Web 工具，Kafka Manager。这个管理工具可以很容易地发现分布在集群中的哪些 topic 分布不均匀，或者是分区在整个集群分布不均匀的的情况。支持管理多个集群、选择副本、副本重新分配以及创建 Topic。同时，这个管理工具也是一个非常好的可以快速浏览 Kafka 集群的工具

    1. 管理多个 kafka 集群
    2. 便捷的检查 kafka 集群状态 (topics, brokers, 备份分布情况, 分区分布情况)
    3. 选择要运行的副本
    4. 基于当前分区状况进行
    5. 可以选择 topic 配置并创建 topic(0.8.1.1和0.8.2的配置不同)
    6. 删除 topic(只支持0.8.2以上的版本并且要在broker配置中设置 delete.topic.enable=true)
    7. Topic list 会指明哪些 topic 被删除（在0.8.2以上版本适用）
    8. 为已存在的 topic 增加分区
    9. 为已存在的 topic 更新配置
    10. 在多个 topic 上批量重分区
    11. 在多个 topic 上批量重分区(可选 partition broker 位置)

    kafka-manager 项目地址：https://github.com/yahoo/kafka-manager

    2.安装配置

    kafka-manager 只是一个 kafka 集群的管理工具，只需要在一台机器上安装即可，选择资源比较充足的机器安装.

    2.1. 环境要求

    1. jdk 1.8
    2. 三台 kafka 集群
    3. 系统 centos 7
