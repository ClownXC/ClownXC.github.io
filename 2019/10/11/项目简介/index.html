滴滴智慧出行项目简介

基于滴滴乘车出行真实项目，利用滴滴出行真实脱敏数据构建大数据智慧出行平台

1. 功能

  1. 智慧出行网约车订单及轨迹监控

    使用 Flume 集成 Kafka 收集轨迹数据，存储至 HBase 中，通过 SparkStreaming 实时统计订单数据和乘车人数，Redis 作为中间层实时缓冲统计数据。

  2. 基于 SparkSQL 自定义 HBase 数据源

    自定义数据源对比原生数据源增加了列剪枝谓词下推，优化了性能。同时支持 Blukload方式向 HBase 存储数据

  3. 智慧出行数据接收和落地

    随着 MySQL 业务数据库数据量增大,性能下降，通过 Maxwell 收集 binlog 日志，将解析之后的 Json 数据发送至Kafka 中通过 SparkStreaming 实时消费 Kafka 数据，将数据实时同步至 HBase 中进行统计分析。

  4. Spark 离线与实时任务监控，实现细粒度任务的监控和异常报警

  5. 智慧交通数据大屏之用户统计分析、订单数据监控

    离线统计订单表汇总、用户数据、活跃数据留存以及区域订单热力图实时分析

2. 代码地址
